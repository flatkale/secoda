## **How to use "questions" in Secoda**

Use the Questions feature as a place for team members to make data requests, ask questions about the data, and then, make it possible for others to view the ensuing responses, even after the question has been answered.

![](https://3544070331-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2FqxbfBjqa19PllrcxgKxO%2Fuploads%2FiUlBj23fqKLXIN6KzloU%2FScreen%20Shot%202022-04-08%20at%2012.34.50%20PM.png?alt=media&token=2b49f4d8-c0ce-4f92-ae94-f2826fe59212)

If Slack is connected to your Secoda workspace, you can ask and answer questions directly from there.
