Endpoint

https://api.secoda.co/dictionary/terms/

Method

GET

### Create a New Dictionary Term

Endpoint

<https://api.secoda.co/dictionary/>

Method

POST

Accepted Body Parameters

name, definition, sql, owner

* Sample Response  
{  
"msg":"Successfully created dictionary term"  
}

* Python Example  
import requests  
  
headers \= {  
"Authorization": "Bearer <Your Key>"  
}  
response \= requests.post(  
"<https://api.secoda.co/dictionary/>",  
 headers\=headers,  
 json\={  
'name': 'Get Users'  
'definition': 'This is the definition of \[Google\](<https://google.com>)',  
'sql': 'SELECT \* FROM tickets.users',  
'owner': "kash@secoda.co"  
}  
)  
print(response.json())

### Update Dictionary Term

Endpoint

<https://api.secoda.co/dictionary/><:id>

Method

PUT

Accepted Body Parameters

name, definition, sql, owner

* Sample Response  
{  
"msg":"Successfully updated dictionary term"  
}

* Python Example  
import requests  
  
headers \= {  
"Authorization": "Bearer <Your Key>"  
}  
response \= requests.put(  
"<https://api.secoda.co/dictionary/0krl0y8cvt92uvo2>",  
 headers\=headers,  
 json\={"name": "New Name"}  
)  
print(response.json())
