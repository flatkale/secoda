## Invite a New User

<Note>
The **Endpoint** field for self-hosted instances is `<host_url>/api/v1/dictionary/invite/`For example, https://secoda.company.com/api/v1/user/invite/
</Note>

### Parameters

<ParamField body="email" required type="string">
  The email of the user you're inviting.
</ParamField>

<ParamField body="role" type="string">
  The role of the user you're inviting.
</ParamField>

<RequestExample>

```python Python Example
import requests

headers = {
"Authorization": "Bearer <Your Key>"
}
response = requests.post(
"<https://api.secoda.co/user/invite/>",
headers=headers,
json={
'email': 'kash@secoda.co',
'role': 'Admin'
}
)
print(response.json())
```

</RequestExample>
