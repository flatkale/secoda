## Invite a New User

Endpoint

`https://api.secoda.co/user/invite/`

Method

POST

Accepted Body Parameters

email, role

The **Endpoint** field for self-hosted instances is `\<host`\_`url>/api/v1/dictionary/invite/`For example, https://secoda.company.com/api/v1/user/invite/

```json Sample Response
{
  "msg": "Invitation sent to kash@secoda.co"
}
```

```python Python Example
import requests

headers \= {
"Authorization": "Bearer <Your Key>"
}
response \= requests.post(
"<https://api.secoda.co/user/invite/>",
 headers\=headers,
 json\={
'email': 'kash@secoda.co',
'role': 'Admin'
}
)
print(response.json())
```
